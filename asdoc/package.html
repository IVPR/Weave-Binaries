lang()function 
public function lang(text:String, ... parameters):String

This is a convenient global function for retrieving localized text. Sample syntax: lang("hello world") You can also specify a format string with parameters which will be passed to StringUtil.substitute(): lang("{0} and {1}", first, second)

Parameters

text:String — The original text or format string to translate.
 
... parameters — Parameters to be passed to StringUtil.substitute() if the text is to be treated as a format string.

Returns
String
