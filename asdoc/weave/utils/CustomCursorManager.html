registerEmbeddedCursor()method 
public static function registerEmbeddedCursor(name:String, bitmapAssetClass:Class, xHotSpot:Number, yHotSpot:Number):void

This will register an embedded cursor. To embed a cursor in your own class, follow this example: public static const MY_CURSOR:String = "myCursor"; [Embed(source="/weave/resources/images/myCursor.png")] private static var myCursor:Class; CustomCursorManager.registerEmbeddedCursor(MY_CURSOR, myCursor, 0, 0);

Parameters

name:String — A name for the cursor.
 
bitmapAssetClass:Class — The Class containing the embedded cursor image.
 
xHotSpot:Number — The X coordinate of the hot spot. Set to NaN to use the center X coordinate.
 
yHotSpot:Number — The Y coordinate of the hot spot. Set to NaN to use the center Y coordinate.

showCursor()method 
public static function showCursor(name:String):int

This function is to set the cursor to standard cursor types like hand cursor, link cursor, etc. Look at the static String constants to get all the types of available cursors.

Parameters

name:String — The name of a registered cursor.

Returns
int — An id mapped to the cursor that can be passed to removeCursor() later.
