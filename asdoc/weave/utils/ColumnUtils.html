joinColumns()method 
public static function joinColumns(columns:Array, dataType:Class = null, allowMissingData:Boolean = false, keyFilter:Object = null):Array

This function takes the common keys from a list of columns and generates a table of data values for each key from each specified column.

Parameters

columns:Array — A list of IAttributeColumns to compute a join table from.
 
dataType:Class (default = null) — The dataType parameter to pass to IAttributeColumn.getValueFromKey().
 
allowMissingData:Boolean (default = false) — If this is set to true, then all keys will be included in the join result. Otherwise, only the keys that have associated values will be included.
 
keyFilter:Object (default = null) — Either an IKeyFilter or an Array of IQualifiedKey objects used to filter the results.

Returns
Array — An Array of Arrays, the first being IQualifiedKeys and the rest being Arrays data values from the given columns that correspond to the IQualifiedKeys.
