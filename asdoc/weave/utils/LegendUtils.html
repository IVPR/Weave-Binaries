getBoundsFromItemID()method
public static function getBoundsFromItemID(fullScreenBounds:IBounds2D, index:int, outputScreenBounds:IBounds2D, totalItemCount:int, maxColumns:int = 1, transposeInRow:Boolean = false):void

This function will fill in outputScreenBounds with the bounds of the item, relative to the legend placement.

Parameters

fullScreenBounds:IBounds2D — The full bounds of the screen.
 
index:int — The index of the item relative to totalItemCount.
 
outputScreenBounds:IBounds2D — The output bounds in screen coordinates.
 
totalItemCount:int — The total number of items in the legend.
 
maxColumns:int (default = 1) — The maximum number of columns. When a column is filled, the next item is placed at the beginning of the next row.
 
transposeInRow:Boolean (default = false) — If true, then the column of the item will be transposed inside the row. For example, with maxColumns = 5 (columns 0 .. 4), an item in column 0 would be transposed to column 4. An item in column 3 would be transposed to column 1, etc.

Parameters

verticalAlign:String (default = middle)
