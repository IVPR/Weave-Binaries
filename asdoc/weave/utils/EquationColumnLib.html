Packageweave.utils
Classpublic class EquationColumnLib
InheritanceEquationColumnLib Object

This class contains static functions that access values from IAttributeColumn objects. Many of the functions in this library use the static variable 'currentRecordKey'. This value should be set before calling a function that uses it.
