VectorUtilsMethods
 
Packageweave.utils
Classpublic class VectorUtils
InheritanceVectorUtils Object

This class contains static functions that manipulate Vectors and Arrays. Functions with as parameter types support both Vector and Array. Vector.<rt; is not used because it causes compiler errors.
