public static function drawCenteredIcon(graphics:Graphics, x:Number, y:Number, source:BitmapData, fillColor:Number, fillAlpha:Number = 1.0, scale:Number = 1.0):void

This function will draw an image centered on an x,y coordinate.

Parameters

graphics:Graphics — The Graphics object where the icon should be drawn.
 
x:Number — The center x coordinate used to draw the icon.
 
y:Number — The center y coordinate used to draw the icon.
 
source:BitmapData — The icon image.
 
fillColor:Number (default = NaN) — A background fill color.
 
fillAlpha:Number (default = 1.0) — A background fill alpha.
 
scale:Number (default = 1.0) — A scale value used to resize the source image.
