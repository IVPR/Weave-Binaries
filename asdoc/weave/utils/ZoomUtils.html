conformDataBoundsToAspectRatio()method
public static function conformDataBoundsToAspectRatio(dataBoundsToModify:IBounds2D, screenBounds:IBounds2D, aspectRatio:Number):void

Enforce an aspect ratio on xDataUnitsPerPixel to yDataUnitsPerPixel. This will increase the size of the given dataBounds either vertically or horizontally if necessary.

Parameters

dataBoundsToModify:IBounds2D — The Bounds2D object to enforce an aspectRatio on.
 
screenBounds:IBounds2D — The Bounds2D object the dataBounds will be mapped to.
 
aspectRatio:Number — The aspect ratio to enforce (absolute value will be used).

public static function getScaleFromZoomLevel(fullDataBounds:IBounds2D, minScreenSize:Number, zoomLevel:Number):Number

This function calculates the unambiguous scale value (defined as pixels per data unit) based on the parameters.

Parameters

fullDataBounds:IBounds2D — The full extent in data coordinates.
 
minScreenSize:Number — The minimum size that the fullDataBounds can appear as on the screen (the screen size of zoom level zero).
 
zoomLevel:Number — The zoom level, where the screen size of the full extent is 2^zoomLevel minSize.

Returns
Number — The scale, defined as pixels per data unit.
getZoomLevel()method 
public static function getZoomLevel(dataBounds:IBounds2D, screenBounds:IBounds2D, fullDataBounds:IBounds2D, minScreenSize:Number):Number

This function calculates the zoom level. If dataBounds is scaled to fit into screenBounds, the screen size of fullDataBounds would be 2^zoomLevel minSize. Zoom level is defined this way to be compatible with the zoom level used by Google Maps and other tiled WMS services.

Parameters

dataBounds:IBounds2D — The visible data coordinates.
 
screenBounds:IBounds2D — The visible screen coordinates.
 
fullDataBounds:IBounds2D — The full extent in data coordinates.
 
minScreenSize:Number — The minimum size that the fullDataBounds can appear as on the screen (the screen size of zoom level zero).

Returns
Number — The zoom level, where the screen size of the full extent is 2^zoomLevel minSize.

zoomDataBoundsByRelativeScreenScale()method 
public static function zoomDataBoundsByRelativeScreenScale(dataBounds:IBounds2D, screenBounds:IBounds2D, xScreenOrigin:Number, yScreenOrigin:Number, relativeScale:Number, repositionCenter:Boolean):void

scale greater than 1 will zoom in, scale less than 1 will zoom out

Parameters

dataBounds:IBounds2D — The visible data bounds
 
screenBounds:IBounds2D — The screen bounds that the data bounds is scaled to fit in
 
xScreenOrigin:Number — The x screen coordinate used as the scale origin
 
yScreenOrigin:Number — The y screen coordinate used as the scale origin
 
relativeScale:Number — The relative scale value used to scale the data bounds
 
repositionCenter:Boolean — If true, this causes the data coordinates to be associated with the screen origin to be moved to the center of the screen bounds after scaling.
