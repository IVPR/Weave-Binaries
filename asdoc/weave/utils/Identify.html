public static function identify(... propertyNames):void

Enables (or disables) tooltips on every DisplayObject, showing properties and debugIds.

Parameters

... propertyNames — An optional list of property names to display in the tooltips. If none specified, tooltips will be toggled with default properties. To disable tooltips, pass a single parameter: false.
