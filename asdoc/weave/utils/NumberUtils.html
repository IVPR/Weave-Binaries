getNumberFromNumberOrPercent()method 
public static function getNumberFromNumberOrPercent(numberOrPercent:String, wholeForPercentage:Number):Number

getNumberFromNumberOrPercent This function will convert a String like "75%" into a Number using the calculation "whole percent / 100". If the String does not have a "%" sign in it, it will be treated as an absolute number.

Parameters

numberOrPercent:String — This string can be either a number like "640" or a percentage like "50%".
 
wholeForPercentage:Number — If the 'numberOrPercent' parameter contains the '%' sign, this will be used as the 'whole' value used in the calculation.

Returns
Number — The 'numberOrPercent' parameter as a Number if it does not have a '%' sign, or (whole percent / 100) if it does.
