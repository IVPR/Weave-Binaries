InheritanceNearestNeighborLayoutAlgorithm AbstractLayoutAlgorithm CallbackCollection Object
Implements ILayoutAlgorithm

An implementation of the NEAREST_NEIGHBOR dimensional ordering algorithm. This algorithm finds nearest neighbors of a dimension successfully until all dimensions have been added.
