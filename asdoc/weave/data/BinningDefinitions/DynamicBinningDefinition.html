Packageweave.data.BinningDefinitions
Classpublic class DynamicBinningDefinition
InheritanceDynamicBinningDefinition LinkableDynamicObject LinkableWatcher Object
Implements IBinningDefinition

This provides a wrapper for a dynamically created IBinningDefinition. When generateBinClassifiersForColumn(column) is called, the column will be monitored for changes and results will be computed automatically.

asyncResultCallbacksproperty
asyncResultCallbacks:ICallbackCollection  [read-only]

These callbacks will be triggered when the current asynchronous task completes. The callbacks of the IBinningDefinition will NOT be triggered as a result of these callbacks triggering, so to know when the results change you must add a callback to this particular callback collection.

generateBinClassifiersForColumn()method
public function generateBinClassifiersForColumn(column:IAttributeColumn):void

This will begin an asynchronous task to generate a list of IBinClassifier objects. Only one task can be carried out at a time. The asyncResultCallbacks will be triggered when the task completes.

Parameters

column:IAttributeColumn — A column for which to generate IBinClassifier objects.
