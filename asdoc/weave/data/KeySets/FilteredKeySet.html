Classpublic class FilteredKeySet
InheritanceFilteredKeySet CallbackCollection Object
Implements IFilteredKeySet

A FilteredKeySet has a base set of keys and an optional filter. The resulting set of keys becomes the intersection of the base set with the filter.

Parameters
