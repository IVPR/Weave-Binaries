Implements IKeySet

This provides the keys from an existing IKeySet in a sorted order. Callbacks will trigger when the sorted result changes.

public static function generateSortCopyFunction(columns:Array, sortDirections:Array = null):Function

Generates a function like function(keys:Array):Array that returns a sorted copy of an Array of keys. Note that the resulting sort function depends on WeaveAPI.StatisticsManager, so the sort function should be called again when statistics change for any of the columns you provide.

Parameters

columns:Array — An Array of IAttributeColumns or Functions mapping IQualifiedKeys to Numbers.
 
sortDirections:Array (default = null) — Sort directions (-1, 0, 1)

Returns
