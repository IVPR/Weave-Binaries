Packageweave.data.DataSources
Classpublic class AbstractDataSource_old
InheritanceAbstractDataSource_old AbstractDataSource Object
Implements IDataSource_old
Subclasses TransposedDataSource, WeaveDataSource, WFSDataSource, XLSDataSource

This is a base class to make it easier to develope a new class that implements IDataSource_old. getHierarchyRoot() and generateHierarchyNode() should be overridden if the data source does not use XMLEntityNode to build its hierarchy.

protected function requestHierarchyFromSource(subtreeNode:XML = null):void

This function must be implemented by classes by extend AbstractDataSource. This function should make a request to the source to fill in the hierarchy at the given subtree. This function will only be called once per subtreeNode.
