public function CSVParser(asyncMode:Boolean = false, delimiter:String = ,, quote:String = ")

Creates a CSVParser.

Parameters
asyncMode:Boolean (default = false) — If this is set to true, parseCSV() will work asynchronously and trigger callbacks when it finishes. If this is set to false, no callback collection will be generated for this instance of CSVParser as a result of calling its methods. Note that if asyncMode is enabled, you can only parse one CSV string at a time.
 
delimiter:String (default = ,) — A single character for the delimiter
 
quote:String (default = ") — A single character for the quote

public function convertRecordsToRows(records:Array, columnOrder:Array = null, allowBlankColumns:Boolean = false, headerDepth:int = 1):Array

This function converts an Array of Objects (compatible with DataGrid) to an Array of Arrays compatible with other functions in this class.

Parameters

records:Array — An Array of Objects containing String properties.
 
columnOrder:Array (default = null) — An optional list of column names to use in order. Must be a two-dimensional Array if headerDepth > 1.
 
allowBlankColumns:Boolean (default = false) — If this is set to true, then the function will include all columns even if they are blank.
 
headerDepth:int (default = 1) — The depth of record properties. If depth is greater than one, the records will be treated as nested objects.

Returns
Array — An Array of Arrays, the first being a header line containing all the property names.
