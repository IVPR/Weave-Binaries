Classpublic class WeaveAdminService
InheritanceWeaveAdminService Object
Implements IWeaveEntityManagementService, flash.events.IEventDispatcher

The functions in this class correspond directly to Weave servlet functions written in Java. This object uses a queue to guarantee that asynchronous servlet calls will execute in the order they are requested.
