Packageweave.core
Classpublic class WeaveXMLEncoder
InheritanceWeaveXMLEncoder mx.rpc.xml.SimpleXMLEncoder

This extension of SimpleXMLEncoder adds support for encoding TypedSessionState objects, XML values, and null values. The static encode() method eliminates the need to create an instance of XMLEncoder.
