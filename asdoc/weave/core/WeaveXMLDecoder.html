Packageweave.core
Classpublic class WeaveXMLDecoder
InheritanceWeaveXMLDecoder mx.rpc.xml.SimpleXMLDecoder

This extension of SimpleXMLDecoder adds support for XML objects encoded with XMLEncoder. The static methods provided here eliminate the need to create an instance of XMLDecoder.
