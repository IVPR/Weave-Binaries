Classpublic class ExternalSessionStateInterface
InheritanceExternalSessionStateInterface Object
Implements IExternalSessionStateInterface

A set of static functions intended for use as a JavaScript API. The user interface in Weave is initially generated from a saved session state. User interactions affect the session state, and changes in the session state affect the display at runtime. The API provides a window into the session state so most interactions that can be made through the GUI can also be made through JavaScript calls.

evaluateExpression()method 
public function evaluateExpression(scopeObjectPathOrVariableName:Object, expression:String, variables:Object = null, staticLibraries:Array = null, assignVariableName:String = null):*

This function will evaluate an expression using the compiler. An object path may be passed as the first parameter to act as the this pointer for the expression, or libraries may be included by passing an array of fully qualified names. Note that any code written for this function depends on the implementation of the ActionScript code inside Weave, which is subject to change.

Parameters

scopeObjectPathOrVariableName:Object — A sequence of child names used to refer to an object appearing in the session state, or the name of a previously saved expression, which will be used as the this pointer when evaluating the expression.
 
expression:String — The expression to evaluate.
 
variables:Object (default = null) — A hash map of variable names to values.
 
staticLibraries:Array (default = null) — An array of fully qualified class names which contain static methods to include for future expressions, including this one.
 
assignVariableName:String (default = null) — An optional variable name to associate with the result of evaluating this expression. Use an empty string ("") to prevent returning a value.

Returns
* — The value of the evaluated expression, or undefined if assignVariableName was specified.

requestObject()method 
public function requestObject(objectPath:Array, objectType:String):Boolean

This function will dynamically create an object at the specified location in the session state if its parent implements ILinkableCompositeObject. If the object at the specified location already exists and is of the requested type, this function does nothing. If the parent of the dynamic object to be created implements ILinkableHashMap, a value of null for the child name will cause a new name to be generated. If the parent of the dynamic object to be created implements ILinkableDynamicObject, the name of the child refers to the name of a static object appearing at the top level of the session state. A child name equal to null in this case will create a local object that does not appear at the top level of the session state.

Parameters

objectPath:Array — A sequence of child names used to refer to an object appearing in the session state. A child index number may be used in place of a name in the path when its parent object is a LinkableHashMap.
 
objectType:String — The qualified name of a class implementing ILinkableObject.

Returns
Boolean — true if, after calling this function, an object of the requested type exists at the requested location.
