public static function drawInvisiblePolygon(sprite:Sprite, coords:Array = null, normalizedCoords:Boolean = false, borderThickness:Number = 0):void

Draws an invisible polygon on a component for catching mouse events.

Parameters

sprite:Sprite — The component.
 
coords:Array (default = null) — An array containing coordinates like [x0,y0, x1,y1, ...]. If omitted, a default rectangle will be used.
 
normalizedCoords:Boolean (default = false) — true for normalized coordinates (x and y values will be multiplied by sprite.width and sprite.height)
 
borderThickness:Number (default = 0) — The thickness of the polygon border in pixels.
