probe()method 
public function probe(queryBounds:IBounds2D):Array

This function will get the objects whose bounds are closest to the center of the queryBounds object. Generally this function will return an array of at most one object. Sometimes, it may return more than one object if there are multiple objects with equivalent distance to the center of the bounds object.

Parameters

queryBounds:IBounds2D — A bounds used to query the spatial index.

Returns
Array — An array of objects.
