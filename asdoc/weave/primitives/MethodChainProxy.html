Classpublic dynamic class MethodChainProxy
InheritanceMethodChainProxy flash.utils.Proxy
Subclasses GraphicsBuffer

This is a Proxy object that wraps function calls in a way that allows method chaining (obj.f(1).g(2).h(3)). Multiple objects can be wrapped in a single MethodChainProxy object. In that case, method calls will use the first object with a matching property name. If a wrapped function returns a value, it can be accessed using valueOf(): obj.f(1).g(2).valueOf() Non-function properties are automatically wrapped in setter/getter functions that will set the value if given one parameter and get the value if given zero parameters.

Parameters
primaryObject:Object — The primary object to be proxied by chainable method wrappers, or null if you wish to specify objects in an Array in the moreObjects parameter.
 
... moreObjects — Additional objects to be checked in order in case properties/methods do not exist on the primary object. You may specify a single Array as the moreObjects parameter if the primary object parameter is set to null.

Examples
     var shape:Shape = new Shape();
     var graphicsChain:MethodChainProxy = new MethodChainProxy(shape, shape.graphics);
     
     this.rawChildren.addChild(shape);
     
     graphicsChain.lineStyle(1,1,1)
      .moveTo(0,0)
      .lineTo(10,10)
      .lineTo(10,0)
      .lineTo(0,0)
      .x(20)
      .y(40)
      .scaleX(3)
      .scaleY(4);
     
     trace( 'Width of shape is:', graphicsChain.width() );
     trace( 'getRect() returns:', graphicsChain.getRect(this).valueOf() );
