public function getColumn(dataSource:IDataSource, metadata:Object):IAttributeColumn

This function will return the same IAttributeColumn for identical metadata values. Use this function to avoid downloading duplicate column data.

Parameters

dataSource:IDataSource — The data source to request the column from if it is not already cached.
 
metadata:Object — The metadata to be passed to dataSource.getAttributeColumn().

Returns
