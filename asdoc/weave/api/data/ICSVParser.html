convertRecordsToRows()method
public function convertRecordsToRows(records:Array, columnOrder:Array = null, allowBlankColumns:Boolean = false, headerDepth:int = 1):Array

This function converts an Array of Objects (compatible with DataGrid) to an Array of Arrays compatible with other functions in this class.

Parameters

records:Array — An Array of Objects containing String properties.
 
columnOrder:Array (default = null) — An optional list of column names to use in order. Must be a two-dimensional Array if headerDepth > 1.
 
allowBlankColumns:Boolean (default = false) — If this is set to true, then the function will include all columns even if they are blank.
 
headerDepth:int (default = 1) — The depth of record properties. If depth is greater than one, the records will be treated as nested objects.

Returns
Array — An Array of Arrays, the first being a header line containing all the property names.

public function convertRowsToRecords(rows:Array, headerDepth:int = 1):Array

This function converts an Array of Arrays to an Array of Objects compatible with DataGrid.

Parameters

rows:Array — An Array of Arrays, the first being a header line containing property names.
 
headerDepth:int (default = 1) — The number of header rows. If the depth is greater than one, nested record objects will be created.

Returns
Array — An Array of Objects containing String properties using the names in the header line.

getRecordFieldNames()method 
public function getRecordFieldNames(records:Array, includeNullFields:Boolean = false, headerDepth:int = 1):Array

This function returns a comprehensive list of all the field names defined by a list of record objects.

Parameters

records:Array — An Array of record objects.
 
includeNullFields:Boolean (default = false) — If this is true, fields that have null values will be included.
 
headerDepth:int (default = 1) — The depth of record properties. If depth is greater than one, the records will be treated as nested objects.

Returns
Array — A comprehensive list of all the field names defined by the given record objects. If headerDepth > 1, a two-dimensional array will be returned.
