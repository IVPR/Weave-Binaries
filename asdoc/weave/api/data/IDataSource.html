hierarchyRefreshproperty
hierarchyRefresh:ICallbackCollection  [read-only]

When explicitly triggered, this will force the hierarchy to be refreshed. This should not be used to determine when the hierarchy is updated. For that purpose, add a callback directly to the IDataSource instead.
