Packageweave.api.primitives
Interfacepublic interface IBounds2D
Implementors Bounds2D

IBounds2D provides a flexible interface to a Rectangle-like object. The bounds values are stored as xMin,yMin,xMax,yMax instead of x,y,width,height because information is lost when storing as width,height and it causes rounding errors when using includeBounds() and includePoint(), depending on the order you include multiple points.
