Packageweave.api.ui
Interfacepublic interface IPlotterWithGeometries extends IPlotter, ILinkableObject
Implementors CirclePlotter, GeometryPlotter, OldParallelCoordinatesPlotter, SimpleParallelCoordinatesPlotter

This interface defines a plotter whose records are geometric objects with special probing and selection. A plotter which implements this interface is subject to polygon containment algorithms during probing and selection.
