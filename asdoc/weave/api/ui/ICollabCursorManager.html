Packageweave.api.ui
Interfacepublic interface ICollabCursorManager

This is an interface for a collaboration cursor manager, which is in charge of rendering collaboration mouse cursors. After any change in the property of a cursor, if nothing occurs after a set amount of time (for example, 5000 milliseconds), the cursor should fade out of view so it does not clutter the display. The next time the position of that cursor is updated, the cursor should fade back into view quickly during the move animation.
