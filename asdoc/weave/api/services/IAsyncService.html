Interfacepublic interface IAsyncService extends ILinkableObject
Implementors LocalAsyncService, Servlet, WebServiceWrapper

This is an interface for an asynchronous service. The invokeAsyncMethod() function invokes an asynchronous method and returns an AsyncToken which you can add IResponder objects to. This is an ILinkableObject so its busy status can be checked and its URL requests will be cancelled when it is disposed.

public function invokeAsyncMethod(methodName:String, methodParameters:Object = null):AsyncToken

This function will invoke an asynchronous method using the given parameters object. When the method finishes, the AsyncToken returned by this function will call its responders.

Parameters

methodName:String — A String to identify which remote procedure to call.
 
methodParameters:Object (default = null) — Either an Array or an Object to use as a list of parameters.

Returns
AsyncToken — An AsyncToken that you can add responders to.
