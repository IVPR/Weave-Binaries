compileObjectToFunction()method 
public function compileObjectToFunction(compiledObject:ICompiledObject, symbolTable:Object, errorHandler:Function, useThisScope:Boolean, paramNames:Array = null, paramDefaults:Array = null, flattenFunctionDefinition:Boolean = true, bindThis:Object = null):Function

This function is for internal use only.

Parameters

compiledObject:ICompiledObject — Either a CompiledConstant or a CompiledFunctionCall.
 
symbolTable:Object — This is a lookup table containing custom variables and functions that can be used in the expression. Multiple lookup tables can be specified in an Array. The values in the lookup tables may be changed outside the function after compiling.
 
errorHandler:Function — A function that takes an Error and optionally returns true if execution should continue, behaving as if the current instruction returned undefined. This may be set to null, which will cause the Error to be thrown.
 
useThisScope:Boolean — If this is set to true, properties of 'this' can be accessed as if they were local variables.
 
paramNames:Array (default = null) — This specifies local variable names to be associated with the arguments passed in as parameters to the compiled function.
 
paramDefaults:Array (default = null) — This specifies default values corresponding to the parameter names. This must be the same length as the paramNames array.
 
flattenFunctionDefinition:Boolean (default = true) — If set to true and the compiledObject represents a function definition, that function definition will be evaluated and returned.
 
bindThis:Object (default = null) — If non-null, the this symbol will be bound to the given value. Otherwise, it will be dynamically determined by how the function is called.

Returns
Function — A Function that takes any number of parameters and returns the result of evaluating the ICompiledObject.

This function compiles an expression into a Function that evaluates using variables from a symbolTable.

paramNames:Array (default = null) — This specifies local variable names to be associated with the arguments passed in as parameters to the compiled function.
 
paramDefaults:Array (default = null) — This specifies default values corresponding to the parameter names. This must be the same length as the paramNames array.

Returns
Function — A Function generated from the expression String.

Error — If the expression is invalid.
