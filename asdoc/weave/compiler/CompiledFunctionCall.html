Packageweave.compiler
Classpublic class CompiledFunctionCall
InheritanceCompiledFunctionCall Object
Implements ICompiledObject

This serves as a structure for storing the information required to make a function call. This is used in the Compiler class to avoid parsing tokens multiple times. To avoid function call overhead, no public functions are not defined in this class.
