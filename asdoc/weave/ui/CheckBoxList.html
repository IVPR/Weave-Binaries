A public bindable "selection" property is provided which is an Array of Boolean values denoting the state of the checkboxes. The items in the selection Array correspond to the items in the dataProvider and the selection Array will automatically be updated to correspond to changes in dataProvider item order.

Default MXML PropertydataProvider
