configureScrollBars()method 
override protected function configureScrollBars():void

This contains a workaround for a problem in List.configureScrollBars relying on a non-working function CursorBookmark.getViewIndex(). This fixes the bug where the tree would scroll all the way from the bottom to the top when a node is collapsed.
